# 调试台参数配置说明

## 📋 参数对比总结

### 1. 老旧配置文件（slice_config.json）

**问题：**
- 包含已废弃的参数（gridStartX, iconSize等）
- 偏移校准参数（forceSquareOffsetX/Y）无法保存（被强制覆盖为0）
- 部分参数值与当前调试台默认值不一致

**主要差异：**
| 参数 | 老JSON值 | 当前默认值 | 图片优化值 |
|------|----------|------------|------------|
| forceSquareOffsetX | 0 | 0 | **-6** ⚠️ |
| forceSquareOffsetY | 0 | 0 | **0** |
| boundsVarianceThresholdCol | 100 | 30 | **100** |
| boundsMinRowHeight | 135 | 20 | **135** |
| boundsMinColWidth | 135 | 20 | **135** |

### 2. 优化配置文件（slice_config_optimized.json）

基于实际调试测试优化的参数配置，已包含偏移校准等关键设置。

## 🔧 修复内容

### ✅ 问题1：偏移校准无法保存

**原因：**
```typescript
// 旧代码：强制覆盖偏移校准参数
mergedParams.forceSquareOffsetX = DEFAULT_PARAMS.forceSquareOffsetX;  // 强制设为0
mergedParams.forceSquareOffsetY = DEFAULT_PARAMS.forceSquareOffsetY;  // 强制设为0
```

**修复：**
移除强制覆盖逻辑，允许用户自定义偏移校准参数并正确保存到 LocalStorage。

### ✅ 问题2：导入配置时废弃参数污染

**原因：**
老旧配置文件包含已废弃的参数，直接合并可能导致污染当前配置。

**修复：**
导入时只保留当前调试台支持的参数，自动过滤废弃参数。

### ✅ 问题3：缺少快速应用优化参数的入口

**修复：**
添加"✨ 应用优化参数"按钮，一键应用经过实际调试测试的优化参数。

- `iconLineGap`: 多行图标线之间的间距

- `minIconsPerLine`: 每行最小图标数量
  - 默认值：5
  - 优化值：5

### 滑动窗口检测参数（已弃用，保留用于兼容性）

- `slidingWindowCols`: 蓝色竖向矩形窗口宽度（M列）

  - 默认值：30

### 边界检测参数（默认使用）
- `useBoundsDetection`: 始终使用边界检测方法
  - 默认值：true
  - 优化值：true
- `boundsWindowHeight`: 边界检测窗口高度（用于检测行）
  - 默认值：5
  - 优化值：5
- `boundsWindowWidth`: 边界检测窗口宽度（用于检测列）
  - 默认值：5
  - 优化值：5
- `boundsVarianceThresholdRow`: 边界检测颜色方差阈值（行检测）
  - 默认值：30
  - 优化值：30
- `boundsVarianceThresholdCol`: 边界检测颜色方差阈值（列检测）
  - 默认值：30
  - 优化值：**100** ⚠️
- `boundsStepSize`: 边界检测步长（像素）
  - 默认值：1
  - 优化值：1

  - 默认值：20
  - 优化值：**135** ⚠️
- `boundsMinColWidth`: 最小列宽（过滤噪声）
  - 默认值：20
  - 优化值：**135** ⚠️

### 图标形状设置
- `forceSquareIcons`: 强制图标为1:1正方形（基于行高度）
  - 默认值：true
  - 优化值：true
- `forceSquareOffsetX`: 1:1强制时的X轴偏移（像素）
  - 默认值：0
  - 优化值：**-6** ⚠️
- `forceSquareOffsetY`: 1:1强制时的Y轴偏移（像素）
  - 默认值：0
  - 优化值：**0**

### 空图标过滤参数
- `filterEmptyIcons`: 是否过滤空图标（没有内容的方框）
  - 默认值：true
  - 优化值：true
- `emptyIconVarianceThreshold`: 判断空图标的方差阈值（低于此值视为空）
  - 默认值：20
  - 优化值：**20**

## 🎯 优化参数说明

### 关键优化项

1. **boundsVarianceThresholdCol: 100**
   - 列检测方差阈值提高到100，减少误检

2. **boundsMinRowHeight: 135**
   - 最小行高提高到135像素，过滤噪声行

3. **boundsMinColWidth: 135**
   - 最小列宽提高到135像素，过滤噪声列

4. **forceSquareOffsetX: -6**
   - X轴偏移校准为-6像素，修正图标位置

## ⚠️ 注意事项

1. **偏移校准参数会实时保存**
   - 调整偏移校准后，参数会自动保存到 LocalStorage
   - 下次打开页面时会自动加载

2. **导入配置时废弃参数会被过滤**
   - 系统会自动忽略废弃参数
   - 如果检测到废弃参数，会提示用户

3. **LocalStorage键名**
   - LocalStorage键名：`wiki_slice_config`
   - 保存位置：浏览器本地存储
   - 清除方法：点击"恢复默认值"按钮

4. **工作台自动读取调试参数**
   - 工作台裁切时会自动读取 LocalStorage 中的参数
   - 确保调试台和工作台使用相同参数

## 🔍 故障排查

### 问题1：偏移校准调整后丢失
- **原因**：已修复，现在会正确保存
- **解决**：重新调整偏移校准，会自动保存

### 问题2：导入配置后参数不对
- **原因**：可能是老旧配置文件包含废弃参数
- **解决**：使用"应用优化参数"按钮或导入优化配置文件

### 问题3：工作台检测结果与调试台不一致
- **原因**：工作台可能使用了旧的 LocalStorage 参数
- **解决**：
  1. 在调试台调整参数
  2. 点击"导出配置"保存
  3. 在工作台重新裁切
